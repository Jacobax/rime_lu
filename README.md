# rime_lu
[鹭](https://github.com/ledao/lufly-im), 词形版鹤, 含官鹤反查, 英文输入等  
适用于中文输入法，即同文无障碍版，同文应该也可，部分小功能不行，如中文同文剪切板不同  
鹭作者通过将鹤形引入词组中，比较完美地解决了官鹤词库太小的局限，较多地保留了双拼的输入习惯，极大地减小了新学的割裂感，好用请到[鹭](https://github.com/ledao/lufly-im)star
